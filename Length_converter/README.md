# Length Converter

A simple open-source React app to convert values between different length units.

## Features
- Convert between meters, kilometers, centimeters, millimeters, inches, feet, yards, and miles
- Input validation with error handling
- Clean and minimal UI

## Tech Stack
- React
- JavaScript
- CSS

## How It Works
- Takes a numeric length input
- Converts the value to meters first
- Converts meters into the selected target unit
